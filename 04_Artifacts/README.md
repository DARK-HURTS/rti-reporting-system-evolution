## Artifacts

This folder contains the evolution of the RTI annual reporting templates from 2021 to 2025.

### Files included:

- 2021_template.docx — Original baseline reporting template  
- 2022_template.docx — Minor structural improvements  
- 2023_template.docx — Major redesign with structured data fields  
- 2024_template.docx — Form-based controlled input system  
- 2025_template.docx — Analytics and compliance-focused template
