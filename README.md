# RTI Annual Reporting System Redesign & Evolution (2021–2025)

---

## Overview

Led and contributed to the multi-year redesign of the RTI Annual Reporting Template used by public institutions. this was done to improve data quality, standardization, compliance tracking, and analytical usability.

---

## Problem Statement

The original reporting template:

- Allowed excessive free-text input, leading to inconsistent data  
- Lacked standardization across institutions  
- Was prone to user modification and deletion of required sections  
- Limited the ability to aggregate and analyze national RTI data  
- Provided minimal insight into performance and compliance  

## Detailed system evolution analysis:  
→ [`02_Analysis/evolution_analysis.md`](./02_Analysis/evolution_analysis.md)

---

## Approach

### Phase 1: Structural Assessment (2021–2022)
- Identified inconsistencies and lack of data structure  
- Introduced basic accountability fields  

### Phase 2: Data Structuring (2023)
- Expanded reporting categories  
- Introduced applicant biodata tracking  
- Improved segmentation of activities and outcomes  

### Phase 3: Data Control & Standardization (2024)
- Designed form-based template to restrict unauthorized edits  
- Standardized responses using controlled formats (e.g., YES/NO fields)  
- Introduced validation and authorization layers  

### Phase 4: Analytics & Compliance Enhancement (2025)
- Added performance metrics (response times)  
- Introduced records management and compliance indicators  
- Enhanced multi-dimensional tracking of institutional readiness  

---

## Key Improvements (Before → After)

| Area | Before | After |
|------|--------|--------|
| Data Structure | Semi-structured | Fully structured |
| Data Integrity | User-editable | Controlled input system |
| Analytics | Limited | Metrics-driven |
| Compliance Tracking | Basic | Multi-dimensional |
| Standardization | Low | High |
| Usability | Inconsistent | Guided form-based |

---

## Impact

- Improved consistency of reports across institutions  
- Enabled aggregation and analysis of national RTI data  
- Reduced data errors and incomplete submissions  
- Strengthened compliance monitoring and evaluation  
- Improved reliability of reporting for policy and decision-making
